

# uAgent's Setup



#### `fund_agent_if_low`

```python
def fund_agent_if_low(wallet_address: str)
```

Checks the uAgent's wallet balance and adds funds if it's below the registration fee.

**Arguments**:

- `wallet_address` _str_ - The wallet address of the uAgent.
  

**Returns**:

  None



#### `register_agent_with_mailbox`

```python
def register_agent_with_mailbox(agent: Agent, email: str)
```

Registers the uAgent on a mailbox server using the provided email.

**Arguments**:

- `agent` _Agent_ - The uAgent object to be registered.
- `email` _str_ - The email address associated with the uAgent.
  

**Returns**:

  None
